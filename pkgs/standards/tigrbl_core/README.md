# tigrbl-core

![PyPI - Downloads](https://static.pepy.tech/badge/tigrbl-core/month) ![Hits](https://hits.sh/github.com/swarmauri/swarmauri-sdk.svg) ![Python Versions](https://img.shields.io/pypi/pyversions/tigrbl-core.svg) ![License](https://img.shields.io/pypi/l/tigrbl-core.svg) ![Version](https://img.shields.io/pypi/v/tigrbl-core.svg)

## Features

- Core specification and primitive contracts for Tigrbl.
- Operational implementations now live in `tigrbl-ops-oltp`.
- Supports Python 3.10 through 3.12.

## Installation

### uv

```bash
uv add tigrbl-core
```

### pip

```bash
pip install tigrbl-core
```

## Usage

Import from the shared package-specific module namespaces after installation in your environment.

## Features

- Modularized Tigrbl atoms components.
- Python 3.10 to 3.12 support.
- Workspace-ready package for Swarmauri monorepo workflows.

## Installation

```bash
uv add tigrbl_atoms
```

```bash
pip install tigrbl_atoms
```

## Usage

```python
import tigrbl_atoms
```

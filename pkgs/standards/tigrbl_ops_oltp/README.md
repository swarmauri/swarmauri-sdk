# tigrbl-ops-oltp

![PyPI - Downloads](https://static.pepy.tech/badge/tigrbl-ops-oltp/month) ![Hits](https://hits.sh/github.com/swarmauri/swarmauri-sdk.svg) ![Python Versions](https://img.shields.io/badge/python-3.10%20%7C%203.11%20%7C%203.12%20%7C%203.13%20%7C%203.14-blue) ![License](https://img.shields.io/pypi/l/tigrbl-ops-oltp.svg) ![Version](https://img.shields.io/pypi/v/tigrbl-ops-oltp.svg)

## Features

- Transactional OLTP operation implementations for Tigrbl.
- Includes canonical CRUD and bulk operation executors.
- Supports Python 3.10 through 3.12.

## Installation

### uv

```bash
uv add tigrbl-ops-oltp
```

### pip

```bash
pip install tigrbl-ops-oltp
```

## Usage

Import operation callables or parameter markers from `tigrbl_ops_oltp.crud`.

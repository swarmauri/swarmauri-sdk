# tigrbl-base

## Features
- Split package for the Tigrbl framework workspace.
- Python 3.10 - 3.12 compatible metadata.
- Designed for composable package-level imports across Tigrbl layers.

## Installation
### uv
```bash
uv add tigrbl-base
```

### pip
```bash
pip install tigrbl-base
```

## Usage
Import this package as part of the Tigrbl layered workflow.

```python
import tigrbl_base
```
